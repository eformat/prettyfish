Most useful Mermaid ER Diagram patterns

  • entity declaration: Declare an entity with optional attributes block
  • string attribute: String type attribute
  • int attribute: Integer type attribute
  • float attribute: Float/decimal type attribute
  • boolean attribute: Boolean type attribute
  • date attribute: Date type attribute (YYYY-MM-DD)
  • datetime attribute: Datetime type attribute (YYYY-MM-DD HH:MM:SS)
  • timestamp attribute: Timestamp type attribute (Unix timestamp)
