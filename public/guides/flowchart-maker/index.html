Most useful Mermaid Flowchart patterns

  • graph / flowchart: Declare a flowchart with direction: TD (top-down), LR (left-right), BT (bottom-top), RL (right-left), TB (top-bottom)
  • rectangle node: Create a rectangular node with text inside
  • rounded node: Create a rounded rectangle node
  • stadium node: Create an oval/stadium-shaped node
  • subroutine node: Create a subroutine rectangle with double brackets
  • cylinder node: Create a cylinder shape for databases
  • circle node: Create a perfect circle node
  • asymmetric node: Create an asymmetric arrow-like node pointing right

hexagon node nodeId{{Text}} Create a hexagon-shaped node
parallelogram node nodeId[/Text/] Create a parallelogram node with forward slash style
parallelogram alt node nodeId[\\Text\\] Create a parallelogram node with backslash style
trapezoid node nodeId[/Text\\] Create a trapezoid node wider at bottom
trapezoid alt node nodeId[\\Text/] Create a trapezoid node wider at top
double circle node nodeId(((Text))) Create a node with double circle border
solid arrow edge A --> B Draw a solid arrow from node A to node B
open line edge A --- B Draw an open line without arrowhead between nodes
dotted arrow edge A -.-> B Draw a dotted arrow between nodes
thick arrow edge A ==> B Draw a thick/bold arrow between nodes
circle arrow edge A --o B Draw an arrow with circle on target end
cross arrow edge A --x B Draw an arrow with cross on target end
bidirectional arrow A <--> B Draw arrows pointing both directions
circle to circle edge A o--o B Draw a line with circles on both ends
cross to cross edge A x--x B Draw a line with crosses on both ends
edge label A -->|Text| B or A -- Text --> B Add a label on an edge between nodes
multi-line edge A --> B B --> C B --> D Chain multiple edges from one or more nodes
subgraph subgraph id[Title] nodes end Group nodes into a subgraph with optional title and direction
subgraph direction subgraph LR id[Title] nodes end Set explicit direction within a subgraph
