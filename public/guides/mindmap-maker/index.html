Most useful Mermaid Mindmap patterns

  • mindmap declaration: Declare a mind map diagram
  • root node: Central node of the mindmap with circle brackets
  • default shape node: Regular node (default shape, inherits from parent)
  • rounded node: Rounded rectangle node shape
  • rectangle node: Square/rectangular node shape
  • circle node: Circle node shape
  • cloud node: Cloud/bubble node shape
  • bang node: Exclamation/bang node shape
