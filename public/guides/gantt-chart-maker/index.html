Most useful Mermaid Gantt patterns

  • title: Set the chart title displayed at the top
  • dateFormat: Set the date format pattern for parsing task dates
  • axisFormat: Set the format for displaying dates on the axis
  • tickInterval: Set the interval for time axis ticks
  • todayMarker: Style the vertical line marking today's date
  • section: Group tasks into a labeled horizontal section/row
  • simple task: Basic task with start date and duration
  • task with id: Task with identifier for dependencies
