Most useful Mermaid Git Graph patterns

  • gitGraph declaration: Declare a git commit graph diagram
  • commit simple: Add a commit to the current branch
  • commit with message: Commit with custom ID/message
  • commit with tag: Commit with version tag label
  • commit type NORMAL: Regular commit (default type)
  • commit type REVERSE: Reverse/undo commit visualization
  • commit type HIGHLIGHT: Highlighted/important commit
  • branch creation: Create and checkout a new branch
