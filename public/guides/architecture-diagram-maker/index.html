Most useful Mermaid Architecture patterns

  • architecture-beta: Declares an architecture diagram (beta)
  • service: Defines a service node with an icon shape and label
  • group: Groups services together visually
  • in: Places a service inside a group
  • junction: A connection point for routing edges through
  • edge (-->): Connects two services. Sides: T (top), B (bottom), L (left), R (right)
  • edge (<-->): Bidirectional connection between services
  • icon shapes: Built-in icon shapes for service nodes
